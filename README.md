## Installation & Usage

### Install Dependencies & Run (client & server)

```
# Run client (:3000)
cd client
npm install
npm start

# Run server (:9000)
cd server
npm install
npm start

# Run frontend (:3000) & backend (:9000)
cd server
npm run dev
```

### Database

This application uses ```notarealdb``` instead of real databases. You can find users and messages in server/data folder.
